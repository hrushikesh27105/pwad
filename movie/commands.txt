Create a Node.js Movie Ticket Booking Application From Scratch

We will create a movie ticket booking application using:

Node.js
Express.js
HTML
Bootstrap
STEP 1 : Create Project Folder

Create a folder:

movie-booking-app

Open it in:

VS Code
STEP 2 : Open Terminal

In VS Code:

Terminal → New Terminal
STEP 3 : Initialize Node.js Project

Run:

npm init -y

This creates:

package.json
STEP 4 : Install Express

Run:

npm install express

This creates:

node_modules
package-lock.json
STEP 5 : Create Files and Folder

Create:

server.js

Create folder:

public

Inside public, create:

index.html
Final Folder Structure
movie-booking-app
│
├── node_modules
│
├── public
│      └── index.html
│
├── package.json
│
├── package-lock.json
│
└── server.js
STEP 6 : Write server.js Code
File: server.js
const express = require("express");

const path = require("path");

const app = express();

const PORT = 3000;

// serve static files
app.use(express.static(path.join(__dirname, "public")));

// start server
app.listen(PORT, () => {

    console.log(`Server running at http://localhost:${PORT}`);

});
